What are requirements for files uploaded?

There are two files that you can upload.

  1. The first one is the concentration data.
    • This file is required.
    • This file must be in CSV format.
    • The first two columns are coordinates (x and y).
    • The next columns are chemicals and their concentrations: Benzene, CH4, H2S, Toluene, VO, m.Xylene, and p.Xylene.
  2. The second one is the wind data.
    • This file is not required.
    • This file must be in XLSX format or Excel format.

How can I upload files?

  1. Go to the Home page.
  2. Click the box to choose a file, or drag a file into the box.
  3. Click Submit button

How can I edit the visualization?

  1. Click Project icon on the left-hand side.
  2. Click 'KML file' row.
  3. As of now, you can see the chemicals' titles in your KML file. On the right-hand side of each title, there are three options:
    • Edit feature - Change title, width, or color.
    • Hide feature / Show feature - Show or hide the visualization of that chemical.
    • Delete - Remove the visualization of that chemical .

How can I tilt or rotate the map?

Click the 3D button at the low-right corner.
